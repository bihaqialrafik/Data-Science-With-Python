Exploratory Data Analysis sangat penting untuk dilakukan sebelum kita melakukan analisis. ini digunakan untuk melakukan investigasi awal pada data
untuk menemukan pola, untuk menemukan anomali, untuk menguji hipotesis dan untuk memeriksa asumsi dengan bantuan statistik ringkasan dan 
representasi grafis. Dengan melakukan EDA, kita dapat lebih memahami kondisi dataset yang kita miliki. Sehingga, kita dapat memulai pembentukan
model Machine Learning dengan lebih baik kedepannya. 'Memahami kondisi dataset' dapat merujuk pada sejumlah hal termasuk namun tidak terbatas pada poin poin berikut : 

1. Mengekstrak variabel penting dan meninggalkan variabel yang tidak berguna

2. Mengidentifikasi pencilan(outliers), nilai yang hilang(missing values), atau kesalahan manusia (human error)

3. Memahami hubungan, atau kekurangan, antar variabel

4. Pada akhirnya, memaksimalkan wawasan yang kita miliki atas kondisi data dan meminimalkan potensi kesalahan di kemudian hari
